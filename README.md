# Dog Breed Data Analysis 

This project focuses on analyzing dog breed data to understand patterns and insights such as breed characteristics, popularity, and trends.

## Project Overview
The goal of this project is to explore and analyze dog breed data using Python and visualize the results through charts and dashboards.

##  Tools & Technologies
- Python (Pandas, Matplotlib)
- Power BI (for dashboard visualization)
- CSV Dataset

##  Key Analysis
- Distribution of dog breeds
- Most common dog breeds
- Breed characteristics comparison
- Data trends and patterns

##  Output
- Cleaned dataset
- Data analysis using Python
- Interactive dashboard (Power BI)

## Files
- `analysis.py` – Data cleaning and analysis
- `dashboard` – Visualization output

##  What I Learned
- Data cleaning and preprocessing
- Data visualization techniques
- Extracting insights from real-world datasets
